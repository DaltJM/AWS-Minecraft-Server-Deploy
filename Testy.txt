Git commit and push workflow.

1. Make changes to project.
2. Open GitKraken.
3. Stage files with changes.
4. Add a commit message.
5. Commit.
6. Push.
7. If unsure, check GitHub to see if changes have been made.

Git branching workflow.
1. Click 'Branch'
2. Name your branch - No spaces.Make the name descriptive of what you intend to do
3. Make changes to the project. 
4. When finished work on branch, merge it into 'Master'
